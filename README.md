# Rotary Numeric Spinner

A Wokwi/Arduino rotary encoder numeric spinner project.

Original Wokwi project:

https://wokwi.com/projects/441483281452613633

## Status

The GitHub repository has been created and initialized, but I could not directly extract the Wokwi project files from the public project URL in this environment.

To complete the extraction, open the Wokwi project, choose **Download Project ZIP**, then upload that ZIP here. I can then add the real `sketch.ino`, `diagram.json`, `libraries.txt`, and any `.cpp`/`.h` source files to this repository.
